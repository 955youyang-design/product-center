## 2. 关键状态

| 分类 | 枚举值 | 说明 |
|---|---|---|
| 产品主档 | draft / ready / published | 草稿、待补齐、资料完整；published 仅表示产品资料完整，不代表渠道商品已上架 |
| 导入任务 | draft / validating / validate_failed / ready_to_execute / executing / completed / execute_failed | 导入状态流转 |
| 渠道商品 | draft / ready / on_shelf / off_shelf / locked | 渠道商品售卖状态 |
| 商品详情页使用状态 | current / history | 当前使用版本与历史版本 |
| 库存同步 | idle / pending / syncing / success / failed | 库存同步状态 |
| 配置状态 | draft / active / inactive | 店铺配置状态 |
| 素材类型 | main / detail / sku | 素材分类 |
| 素材绑定对象 | product / channel_item / item_detail_page / channel_item_sku | 素材归属对象类型 |
| 客户状态 | enabled / disabled | 客户可用状态 |
| 用户状态 | enabled / disabled | 中台用户端用户和管理端用户的登录可用状态 |
| 中台用户端用户角色 | customer_admin / customer_operator / customer_readonly | 中台用户端固定角色枚举 |
| 管理端用户角色 | super_admin / operation_admin / system_admin / readonly_admin | 中台管理端固定角色枚举 |
| 映射状态 | active / inactive | 类目映射和属性值映射是否生效 |
