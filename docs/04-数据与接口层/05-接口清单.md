## 2. 当前接口约定

- 列表接口统一支持分页、关键字搜索和多条件筛选。
- `PUT` 和执行类 `POST` 接口都必须写入审计信息。
- 商品详情页上传接口必须返回 `detail_page_id`、当前使用标记结果和新增素材引用清单。
- 商品详情页保存接口必须返回最新内容版本、当前使用标记结果和最近编辑信息。
- 商品详情页素材复用接口必须返回新增引用关系和素材当前被多少个详情页使用。
- 商品详情页相关写接口必须校验同一 `channel_item_id` 下最多 3 条详情页记录。
- 导入任务相关接口必须返回当前状态、错误数量、警告数量和最近操作时间。
- 类目动态适配读取接口必须返回中台管理端映射版本标识，便于定位规则来源。
- 中台管理端中，中台用户端用户、类目映射和属性值映射接口必须校验 `customer_id` 权限边界。
- 中台管理端管理端用户接口不允许通过 `customer_id` 过滤或归属。
- 映射管理写接口必须校验同一客户下 active 映射唯一性。
