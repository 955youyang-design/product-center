## 1. 权限测试

| test_case_id | 场景 | 前置条件 | 验证点 | 预期结果 |
|---|---|---|---|---|
| TC-PERM-001 | 无导入权限访问产品导入页 | 使用无导入权限账号登录 | 菜单可见性、路由访问、操作按钮 | 菜单不可见或进入后无权限提示，不能执行上传和校验 |
| TC-PERM-002 | 无编辑权限访问渠道商品详情页 | 使用只读渠道运营账号登录 | 字段只读、保存按钮 | 字段只读且保存按钮隐藏 |
| TC-PERM-005 | 无详情页编辑权限访问渠道商品详情页列表页 | 使用仅查看权限账号登录 | 上传按钮、编辑工作区、设为当前、素材复用接口 | 列表可查看，但上传、编辑、设为当前和素材复用入口隐藏或接口返回无权限 |
| TC-PERM-003 | 无库存调整权限执行库存调整 | 使用只读库存账号登录 | 调整入口和同步入口 | 列表可查看，但库存调整和同步动作不可执行 |
| TC-PERM-004 | 无配置编辑权限保存店铺配置 | 使用只读管理员账号登录 | 保存按钮、接口拦截 | 页面可查看但不能保存，接口返回无权限 |
| TC-MGMT-PERM-001 | 无客户管理权限执行开客户 | 使用只读管理端账号登录 | 开客户按钮、接口拦截 | 客户列表可查看，但不能创建、编辑或停用客户 |
| TC-MGMT-PERM-002 | 无映射管理权限保存映射 | 使用只读管理端账号登录 | 类目映射和属性值映射保存按钮、接口拦截 | 映射列表可查看，但新增、编辑和启停用不可执行 |
| TC-MGMT-PERM-003 | 无系统管理权限重置管理端用户密码 | 使用运营管理员账号登录 | 重置密码入口、接口拦截 | 不能重置管理端用户密码 |
| TC-MGMT-PERM-004 | 跨客户访问中台用户端用户 | 使用只允许访问客户 A 的账号 | 请求客户 B 的中台用户端用户列表或保存接口 | 接口拒绝跨客户访问，页面不展示客户 B 用户 |

## 2. 状态流转测试

| test_case_id | 对象 | 场景 | 验证点 |
|---|---|---|---|
| TC-STATE-001 | 导入任务 | 上传、校验、确认导入 | `draft -> validating -> ready_to_execute -> executing -> completed`；异常时进入 `validate_failed` 或 `execute_failed` |
| TC-STATE-002 | 库存同步 | 触发同步并等待结果 | `idle -> pending -> syncing -> success/failed` |
| TC-STATE-003 | 店铺配置 | 保存草稿、发布生效、停用旧版 | `draft -> active -> inactive` |
| TC-STATE-004 | 渠道商品 | 导入后补齐资料并上下架 | `draft -> ready -> on_shelf/off_shelf`；锁定时进入 `locked` |
| TC-STATE-005 | 商品详情页 | 上传新版本并设为当前使用 | `history/current` 切换正确；同一 `channel_item_id` 同时最多一条 `current`，且详情页总数不超过 3 条 |
| TC-MGMT-STATE-001 | 客户 | 启用、停用 | `enabled -> disabled -> enabled`；停用后阻止新增中台用户端用户和新增 active 映射 |
| TC-MGMT-STATE-002 | 中台用户端用户 | 开用户、停用、启用、重置密码 | `enabled -> disabled -> enabled`；重置密码写审计 |
| TC-MGMT-STATE-003 | 映射关系 | 新增、停用、启用 | `active -> inactive -> active`；同客户 active 唯一性正确 |
| TC-MGMT-STATE-004 | 管理端用户 | 开用户、停用、启用、重置密码 | `enabled -> disabled -> enabled`；不能停用当前登录账号 |
