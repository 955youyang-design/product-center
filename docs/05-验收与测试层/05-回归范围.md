## 1. 必回归页面

- 产品列表、产品详情、产品导入、导入记录
- 商品列表、渠道商品详情页列表、渠道商品详情
- 库存列表
- 配置列表、店铺配置详情
- 中台管理端进入研发时：客户列表、客户详情、类目映射列表、类目映射编辑页、属性值映射列表、属性值映射编辑页、管理端用户列表、管理端用户详情

## 2. 必回归接口

- 导入模板下载、导入上传、导入校验、导入执行、导入任务列表
- 产品详情读取与保存
- 渠道商品详情读取与保存、管理端类目适配结果 查询、商品详情页列表/详情/上传/保存/素材复用
- 库存列表、库存调整、库存同步
- 配置列表、配置详情读取与保存
- 中台管理端进入研发时：客户、中台用户端用户、类目映射、属性值映射、管理端用户相关接口

## 3. 条件性回归

- 新增产品、批量修改、库存详情、库存调整记录、素材管理页面在进入当前迭代时纳入回归。
- 中台管理端当前为V1.0，进入实现迭代时纳入完整回归。
- 历史兼容页面不纳入当前回归。

## 4. 触发回归条件

| 变更类型 | 必回归范围 |
|---|---|
| 导入模板字段、导入校验、商家编码关联调整 | 产品导入、导入记录、主流程导入用例、异常导入用例 |
| 店铺配置规则调整 | 配置列表、店铺配置详情、产品导入、渠道商品详情、配置版本机制相关用例 |
| 商品编辑字段或详情页规则调整 | 商品列表、渠道商品详情页列表、渠道商品详情、管理端类目适配读取及详情页数量上限/素材复用相关用例 |
| 库存同步逻辑调整 | 库存列表、库存相关主流程和异常用例 |
| 中台管理端客户隔离、账号或映射规则调整 | 客户管理、中台用户端用户管理、类目映射、属性值映射、管理端用户管理相关用例 |
