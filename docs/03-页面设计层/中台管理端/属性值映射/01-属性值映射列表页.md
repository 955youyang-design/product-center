## 1. 页面目标

按客户维护渠道属性值到中台标准属性值的映射关系，支持按客户、渠道、类目和属性筛选。

## 2. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 筛选区 | 客户、渠道、类目映射、属性名、属性值、状态 | 当前重点 |
| 表格区 | 展示属性值映射关系、状态和更新时间 | 当前重点 |
| 动作区 | 新增、编辑、启停用 | 当前重点 |

## 3. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 客户 | customer_id / select | 必填筛选条件之一 |
| 渠道 | channel_code / enum | 必填或筛选 |
| 类目映射 | category_mapping_id / select | 建议必填 |
| 渠道属性名 | channel_attr_key / string | 必填 |
| 渠道属性值 | channel_attr_value / string | 必填 |
| 中台标准值 | platform_attr_value / string | 必填 |
| 映射状态 | mapping_status / enum | active / inactive |

## 4. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-MGMT-ATTRIBUTE-MAPPING | 属性值映射 |
| 接口 | API-MGMT-ATTRIBUTE-MAPPING-LIST-GET | 查询属性值映射列表 |
| 接口 | API-MGMT-ATTRIBUTE-MAPPING-STATUS-PUT | 启停用属性值映射 |
