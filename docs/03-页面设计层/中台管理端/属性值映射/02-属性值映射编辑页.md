## 1. 页面目标

新增或编辑某个客户下的渠道属性值到中台标准属性值的映射关系。

## 2. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 基础信息区 | 客户、渠道、类目映射、状态 | 当前重点 |
| 渠道属性区 | 渠道属性名、渠道属性值 | 当前重点 |
| 中台标准值区 | 中台属性和值选择 | 当前重点 |
| 操作区 | 保存、取消、启停用 | 当前重点 |

## 3. 状态与反馈

- 保存前必须确认客户、渠道、类目映射和渠道属性值完整。
- 同一客户下，同一渠道属性值已有 active 映射时，阻止重复新增。
- 类目映射已停用时，不允许新增 active 属性值映射。

## 4. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-MGMT-ATTRIBUTE-MAPPING | 属性值映射 |
| 接口 | API-MGMT-ATTRIBUTE-MAPPING-DETAIL-GET | 查询属性值映射详情 |
| 接口 | API-MGMT-ATTRIBUTE-MAPPING-POST | 新增属性值映射 |
| 接口 | API-MGMT-ATTRIBUTE-MAPPING-PUT | 保存属性值映射 |
