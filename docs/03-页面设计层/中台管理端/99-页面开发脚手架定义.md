## 1. 独立端约束

- 中台管理端是独立端，推荐根路径为 `/management`。
- 不挂入当前中台用户端的 `product-platform` 菜单树。
- 页面设计规范复用当前项目的列表页、详情页、工作页结构。
- React 原型中可通过右下角悬浮入口在中台管理端和中台用户端之间切换；该入口仅为原型演示辅助，不改变独立端和菜单树约束。

## 4. 开发边界

- 中台用户端用户管理在客户详情页签内实现，不单独增加左侧菜单。
- 客户与属性值映射相关接口都必须传递 `customer_id`；类目映射为平台全局规则，不要求 `customer_id`。
- 管理端用户不传 `customer_id`，其权限由管理端角色控制。
