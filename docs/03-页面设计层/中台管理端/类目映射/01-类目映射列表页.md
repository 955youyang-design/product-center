## 1. 页面目标

按渠道统一查看平台全局的渠道类目到商品中台标准类目的映射关系，并发起新增、编辑和启停用。

## 2. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 筛选区 | 渠道、渠道类目、中台类目、状态 | 当前重点 |
| 表格区 | 展示映射关系、状态和更新时间 | 当前重点 |
| 动作区 | 新增、编辑、启停用、跳转属性值映射 | 当前重点 |

## 3. 主次操作

| 动作 | 触发条件 | 结果 |
|---|---|---|
| 新增类目映射 | 选择渠道 | 进入类目映射编辑页 |
| 编辑类目映射 | 点击列表行 | 进入类目映射编辑页 |
| 启停用类目映射 | 有映射管理权限 | 修改映射状态 |
| 查看属性值映射 | 点击行内入口 | 带 `category_mapping_id` 进入属性值映射列表 |

## 4. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 渠道 | channel_code / enum | 必填或筛选 |
| 渠道类目 ID | channel_category_id / string | 必填 |
| 渠道类目名称 | channel_category_name / string | 必填 |
| 中台类目 ID | platform_category_id / string | 必填 |
| 中台类目名称 | platform_category_name / string | 必填 |
| 映射状态 | mapping_status / enum | active / inactive |

## 5. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-MGMT-CATEGORY-MAPPING | 类目映射 |
| 接口 | API-MGMT-CATEGORY-MAPPING-LIST-GET | 查询类目映射列表 |
| 接口 | API-MGMT-CATEGORY-MAPPING-STATUS-PUT | 启停用类目映射 |
