## 1. 页面目标

新增或编辑平台全局的渠道类目到中台标准类目的映射关系。

## 2. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 基础信息区 | 渠道、状态 | 当前重点 |
| 渠道类目区 | 渠道类目 ID、名称、路径 | 当前重点 |
| 中台类目区 | 中台标准类目选择与展示 | 当前重点 |
| 操作区 | 保存、取消、启停用 | 当前重点 |

## 3. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 映射 ID | category_mapping_id / string | 编辑时必填 |
| 渠道 | channel_code / enum | 必填 |
| 渠道类目 ID | channel_category_id / string | 必填 |
| 中台类目 ID | platform_category_id / string | 必填 |
| 状态 | mapping_status / enum | active / inactive |

## 4. 状态与反馈

- 同一 `channel_code + channel_category_id` 已存在 active 记录时，禁止重复新增。
- 停用类目映射前必须提示其下属性值映射可能不可用。
- 保存成功后返回列表并保留渠道筛选条件。

## 5. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-MGMT-CATEGORY-MAPPING | 类目映射 |
| 接口 | API-MGMT-CATEGORY-MAPPING-DETAIL-GET | 查询类目映射详情 |
| 接口 | API-MGMT-CATEGORY-MAPPING-POST | 新增类目映射 |
| 接口 | API-MGMT-CATEGORY-MAPPING-PUT | 保存类目映射 |
