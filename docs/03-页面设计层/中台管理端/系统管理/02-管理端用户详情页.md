## 1. 页面目标

维护中台管理端后台用户的基础资料、角色和账号状态。

## 2. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 基础信息区 | 登录账号、姓名、手机号、邮箱 | 当前重点 |
| 角色区 | 固定角色枚举选择 | 当前重点 |
| 状态区 | 启用、停用、重置密码 | 当前重点 |
| 审计摘要区 | 创建时间、最近登录、最近操作人 | 建议保留 |

## 3. 状态与反馈

- 超级管理员账号停用必须二次确认。
- 用户不能停用自己当前登录账号。
- 角色变更、停用和重置密码必须写审计日志。

## 4. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-MGMT-ADMIN-USER | 管理端用户 |
| 接口 | API-MGMT-ADMIN-USER-DETAIL-GET | 查询管理端用户详情 |
| 接口 | API-MGMT-ADMIN-USER-PUT | 保存管理端用户详情 |
