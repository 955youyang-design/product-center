## 1. 页面目标

管理中台管理端自身后台用户，支持开用户、编辑、启停用和重置密码。

## 2. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 筛选区 | 用户名、姓名、角色、状态 | 当前重点 |
| 表格区 | 展示账号、姓名、角色、状态、最后登录时间 | 当前重点 |
| 动作区 | 开用户、编辑、启停用、重置密码 | 当前重点 |

## 3. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 用户 ID | admin_user_id / string | 系统主键 |
| 登录账号 | login_account / string | 必填，管理端全局唯一 |
| 用户姓名 | user_name / string | 必填 |
| 角色 | admin_user_role / enum | super_admin / operation_admin / system_admin / readonly_admin |
| 用户状态 | user_status / enum | enabled / disabled |
| 最后登录时间 | last_login_at / datetime | 只读 |

## 4. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-MGMT-ADMIN-USER | 管理端用户 |
| 接口 | API-MGMT-ADMIN-USER-LIST-GET | 查询管理端用户列表 |
| 接口 | API-MGMT-ADMIN-USER-CREATE-POST | 开管理端用户 |
| 接口 | API-MGMT-ADMIN-USER-STATUS-PUT | 启停用管理端用户 |
| 接口 | API-MGMT-ADMIN-USER-PASSWORD-POST | 重置管理端用户密码 |
