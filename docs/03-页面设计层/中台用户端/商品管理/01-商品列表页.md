## 1. 页面目标

统一查看已导入商品在不同渠道、店铺下的售卖信息，并为详情页内容管理提供统一跳转入口。

## 2. 入口、出口与前置条件

| 项目 | 内容 |
|---|---|
| 入口 | 菜单 `menu.channel_item` |
| 出口 | 渠道商品详情、渠道商品详情页列表 |
| 前置条件 | 商品查看权限 |

## 3. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 筛选区 | 按渠道、店铺、产品、产品ID、ISBN、商品ID、商品状态、上下架状态筛选 | 当前重点 |
| 统计区 | 展示商品总数、待完善商品数、已上架商品数 | 建议保留 |
| 列表区 | 展示渠道商品标题、标题下方商品ID、关联产品、产品标题下方产品ID、ISBN、价格、类目、上下架状态、库存摘要、详情页数量 / 当前版本摘要 | 当前重点 |
| 列表操作区 | 承接详情跳转、详情页跳转、库存跳转和当前页状态反馈 | 当前重点 |

## 4. 主次操作

| 动作 | 触发条件 | 结果 |
|---|---|---|
| 查看详情 | 点击标题或详情按钮 | 进入渠道商品详情页 |
| 查看详情页 | 点击详情页数量或详情页按钮 | 进入渠道商品详情页列表并带商品筛选条件 |
| 跳转库存 | 点击库存摘要 | 进入库存列表并带筛选条件 |

## 5. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 渠道 | channel_code / 下拉 | 支持淘宝、拼多多、抖音、快手 |
| 店铺 | shop_code / 下拉 | 支持按店铺过滤 |
| 上下架状态 | shelf_status / 下拉 | 全部、已上架、已下架 |
| 商品标题 | item_title / 列表字段 | 支持详情页跳转 |
| 商品ID | item_code / 列表字段 | 渠道商品业务标识，展示在渠道商品标题下方 |
| 产品ID | product_id / 列表字段 | 关联产品主档的系统内部标识，展示在关联产品标题下方 |
| ISBN | isbn / 列表字段 | 从关联产品主档读取并独立展示，支持筛选 |
| 详情页数量 | detail_page_count / 列表字段 | 支持跳转渠道商品详情页列表；单个渠道商品最多 3 条 |

## 6. 状态与反馈

- 未命中数据时展示空态和可返回筛选的引导。
- 勾选条目时显示已选数量，供后续V1.0动作扩展使用。

## 7. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-CHANNEL-ITEM | 渠道商品 |
| 接口 | API-ITEM-LIST-GET | 获取商品列表 |

## 8. 范围边界

- 当前列表页不直接编辑商品字段，单条编辑在详情页承接；商品详情页内容、版本和素材复用统一在渠道商品详情页列表承接；批量修改作为V1.0二级入口独立承接。
