## 1. 页面目标

轻量维护导入后的渠道商品标题、价格、类目属性、销售属性、运费模板、上下架状态、发货时效，并编辑当前使用的商品详情内容。

## 2. 入口、出口与前置条件

| 项目 | 内容 |
|---|---|
| 入口 | 商品列表点击详情 |
| 出口 | 商品列表、渠道商品详情页列表、库存列表 |
| 前置条件 | 渠道商品已存在且具备查看权限 |

## 3. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 摘要区 | 展示产品、渠道、店铺、商家编码、商品状态 | 当前重点 |
| 基础售卖区 | 标题、价格、上下架状态、发货时效 | 当前重点 |
| 类目与属性区 | 读取中台管理端维护的类目/属性映射结果；类目映射按平台全局生效 | 当前重点 |
| 履约区 | 运费模板、发货承诺、售卖状态 | 当前重点 |
| 商品主图区 | 展示当前产品主图；优先读取素材库主图，没有则回退产品封面 | 当前重点 |
| 商品详情编辑区 | 编辑当前使用详情页名称、图文模块、模块顺序和文案 | 当前重点 |
| 详情页关联区 | 展示已关联详情页数量、当前使用详情页和跳转入口 | 当前重点 |
| 关联库存区 | 查看关联 SKU 库存摘要并跳转库存页面 | 建议保留 |

## 4. 主次操作

| 动作 | 触发条件 | 结果 |
|---|---|---|
| 保存商品 | 修改字段后 | 保存渠道商品详情 |
| 刷新类目规则 | 切换店铺或类目后 | 重新读取中台管理端映射结果 |
| 查看详情页 | 点击详情页关联摘要 | 跳转渠道商品详情页列表并带当前商品筛选条件 |
| 保存商品详情 | 编辑当前详情页模块后 | 保存当前使用详情页的名称、模块顺序和文案 |
| 查看库存 | 点击库存摘要 | 跳转库存列表 |

## 5. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 商品标题 | item_title / 输入框 | 必填 |
| 价格 | sale_price / 数字输入 | 必填，大于 0 |
| 类目 | category_code / 级联选择 | 必填，受中台管理端类目映射结果限制 |
| 类目属性 | category_attributes / 动态表单 | 按中台管理端映射结果渲染 |
| 销售属性 | sales_attributes / 动态表单 | 按 SKU 维度维护 |
| 运费模板 | freight_template_id / 下拉 | 按店铺配置返回 |
| 上下架状态 | shelf_status / 单选 | 支持上架、下架 |
| 发货时效 | delivery_sla / 下拉或输入 | 必填 |
| 主图 | main_image / 图片展示 | 优先读取素材库主图，没有则回退产品封面 |
| 详情页名称 | detail_page_name / 输入框 | 当前使用详情页名称 |
| 详情页模块 | detail_blocks / 图文模块 | 支持编辑标题、内容、上移、下移和删除 |

## 6. 状态与反馈

- 切换店铺或类目时，失效字段要提示并清空。
- 无编辑权限或商品锁定时，页面只读。
- 保存成功后展示最近同步时间和操作人。
- 当前商品尚未关联详情页时，页面要提示并提供跳转到渠道商品详情页列表的入口。
- 当前页保存商品详情时，只保存当前使用详情页内容；上传新详情页、切换版本和素材复用仍从渠道商品详情页列表进入。

| 接口 | API-ITEM-DETAIL-PUT | 保存详情 |

## 8. 范围边界

- 当前页聚焦导入后商品的单条编辑与规则联动，批量动作由批量修改工作台承接。
- 当前页不维护类目动态适配、类目映射或属性值映射；规则维护必须在中台管理端完成。
- 当前页可以编辑当前使用详情页的图文内容；详情内容上传、版本化、当前版本切换和素材复用统一由渠道商品详情页列表与素材列表承接。
- 库存仍由库存管理统一维护，详情页只展示摘要和跳转入口。
