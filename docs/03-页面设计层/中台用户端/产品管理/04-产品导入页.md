## 1. 页面目标

通过选择导入类型、渠道类目或产品类目生成模板，上传导入包、执行校验并确认导入。当前支持“渠道商品导入”和“产品资料导入”两条路径：渠道商品导入会写入产品、商品、SKU、库存和素材；产品资料导入只维护中台产品主档和素材。

## 2. 入口、出口与前置条件

| 项目 | 内容 |
|---|---|
| 入口 | 二级菜单 `menu.product.import` |
| 出口 | 导入记录、产品列表 |
| 前置条件 | 导入执行权限；渠道商品导入需要存在可用店铺配置；产品资料导入不要求店铺 |

## 3. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 导入类型区 | 选择渠道商品导入或产品资料导入，并说明两者落库范围 | 当前重点 |
| 渠道与类目区 | 渠道商品导入选择单个渠道、该渠道下的已授权店铺和单个渠道类目；产品资料导入只选择中台产品类目 | 当前重点 |
| 模板与上传区 | 展示当前唯一模板结构，下载模板并上传文件或素材包 | 当前重点 |
| 模板下载管理区 | 展示最近下载的模板记录，最多保留 10 个模板，可重新下载或移除 | 当前重点 |
| 校验结果区 | 显示通过数、失败数、失败原因和待处理关联数 | 当前重点 |
| 确认导入区 | 校验通过后确认执行导入 | 当前重点 |

## 4. 主次操作

| 动作 | 触发条件 | 结果 |
|---|---|---|
| 选择导入类型 | 进入页面后 | 确定本次导入是渠道商品导入还是产品资料导入 |
| 选择渠道与类目 | 选择导入类型后 | 渠道商品导入选择单个渠道、该渠道下一个或多个已授权店铺，并选择一个渠道类目；产品资料导入选择中台产品类目 |
| 查看本次导入模板 | 选择渠道与类目后 | 页面展示本次模板文件名和导入范围，不提供模板类型选择 |
| 下载模板 | 导入范围完整后 | 下载包含本次导入范围字段的唯一模板。渠道商品导入模板包含店铺、线上商品ID、线上SKU ID、平台类目属性、售卖字段、库存和素材字段；产品资料导入模板只包含产品主档、ISBN、产品类目、主图和素材字段 |
| 管理已下载模板 | 模板下载后 | 最近下载模板进入模板下载管理区，最多保留 10 个，可重新下载或移除 |
| 上传文件 | 已下载本次导入模板并选择本地文件后 | 创建导入任务并进入草稿状态 |
| 执行校验 | 上传成功后 | 进入校验中并输出校验结果 |
| 确认导入 | 校验通过后 | 执行落库并跳转导入记录 |

## 5. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 导入类型 | import_type / 单选 | 必填；可选渠道商品导入、产品资料导入 |
| 渠道 | channel_code / 多选 | 渠道商品导入必填；决定平台类目属性字段 |
| 店铺 | shop_code / 下拉或模板明细填写 | 渠道商品导入使用；填写后可返回更精确的类目与属性字段 |
| 渠道类目 | channel_category_id / 单选 | 渠道商品导入必填；类目选项来自当前渠道，不同渠道使用各自平台类目和类目属性字段 |
| 产品类目 | product_category_code / 多选 | 产品资料导入必填；使用中台标准产品类目 |
| 模板说明 | template_scope / 只读 | 展示本次导入范围和生成的模板文件名 |
| 模板版本 | template_version / 只读或下拉 | 随配置版本变化 |
| 导入文件 | file_ref / 上传 | 必填；需符合模板结构 |

## 6. 状态与反馈

- 未选择导入类型、渠道、已授权店铺或渠道类目时，渠道商品导入的下载和上传按钮不可点击。
- 产品资料导入未选择类目时，下载和上传按钮不可点击。
- 未下载本次导入模板时，上传按钮不可点击。
- 模板下载管理最多保留 10 个模板；超过 10 个时保留最近下载的 10 个。
- 渠道商品导入校验店铺授权、线上商品ID、线上SKU ID、平台类目属性、售卖字段、库存和商家编码关联。
- 产品资料导入校验产品主档字段、ISBN唯一性、产品类目和素材引用。
- 校验失败时显示错误数量、错误类型和失败行下载入口。
- 存在未处理商家编码关联或 ISBN 冲突时，确认导入按钮禁用并提示先解决问题。

## 8. 范围边界

- 当前页不负责单条产品编辑，编辑在产品详情页完成。
- 产品资料导入不创建渠道商品；渠道商品导入才创建或更新渠道商品、SKU 和库存。
- 当前页只负责模板下载、上传校验和导入落库，不承接导入后的商品编辑。
