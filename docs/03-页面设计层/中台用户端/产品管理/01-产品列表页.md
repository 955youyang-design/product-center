## 1. 页面目标

查看和筛选产品主档信息，作为产品维护总入口。

## 2. 入口、出口与前置条件

| 项目 | 内容 |
|---|---|
| 入口 | 左侧菜单 `menu.product` |
| 出口 | 产品详情、新增产品 |
| 前置条件 | 产品查看权限 |

## 3. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 筛选区 | 按书名/ISBN/商家编码/状态/来源渠道筛选产品 | 当前重点 |
| 统计区 | 展示产品总数、待补充详情数、已关联商品数 | 建议保留 |
| 列表区 | 展示产品基础信息、产品标题下方产品ID、ISBN、商家编码、SKU 摘要、关联商品数、产品库存、最近更新时间 | 当前重点 |
| 快捷操作区 | 查看详情、进入新增产品 | 当前重点 |

## 4. 主次操作

| 动作 | 触发条件 | 结果 |
|---|---|---|
| 搜索/重置 | 输入筛选条件后 | 刷新列表结果 |
| 查看详情 | 点击产品名称或详情按钮 | 进入产品详情页 |
| 新增产品 | 点击新增按钮 | 进入V1.0页面或保留提示 |

## 5. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 关键词 | keyword / 输入框 | 支持书名、ISBN、商家编码模糊搜索 |
| 产品ID | product_id / 列表字段 | 系统内部产品唯一标识，展示在产品标题下方 |
| ISBN | isbn / 列表字段 | 独立列展示，作为产品业务标识 |
| 产品库存 | product_inventory / 列表字段 | 按产品名称聚合库存记录，展示可售库存和总库存 |
| 产品状态 | product_status / 下拉 | 默认全部；资料完整、待补齐、草稿 |
| 来源渠道 | source_channel / 多选 | 支持淘宝、拼多多、抖音、快手 |
| 关联商品数 | item_count / 只读 | 列表展示字段 |

## 6. 状态与反馈

- 初始加载失败时展示重试提示。
- 无数据时展示空态说明，并保留新增产品入口占位。
- 进入详情页后，返回应保留筛选条件和滚动位置。

## 7. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-PRODUCT | 产品主档对象 |
| 接口 | API-PROD-LIST-GET | 获取产品列表 |

## 8. 范围边界

- 当前页不直接编辑产品字段，编辑发生在产品详情页。
- 当前页不承接模板下载，模板下载发生在产品导入页。
