## 1. 页面目标

查看某个产品素材文件夹下的图片内容。进入详情后先展示产品信息和素材统计，再按主图、详情图、SKU 图分文件夹展示图片、文件信息和引用关系。

## 2. 入口、出口与前置条件

| 项目 | 内容 |
|---|---|
| 入口 | 素材列表的产品文件夹进入 |
| 出口 | 素材列表、产品详情、渠道商品详情 |
| 前置条件 | 素材已存在 |

## 3. 页面结构

| 区块 | 说明 | 当前重点 |
|---|---|---|
| 产品信息区 | 展示产品封面、产品名称、产品编码、标准类目和各类型图片数量 | V1.0 |
| 类型文件夹区 | 按主图、详情图、SKU 图分文件夹展示图片内容 | V1.0 |
| 引用关系区 | 每张图片展示引用详情页、引用次数和素材 ID | V1.0 |

## 4. 主次操作

| 动作 | 触发条件 | 结果 |
|---|---|---|
| 返回列表 | 点击返回 | 返回素材列表 |
| 上传到当前产品 | 点击上传到当前产品 | 新图片进入当前产品对应类型文件夹 |
| 复制引用 | 点击单张图片复制引用 | 复制素材 ID，用于详情页复用 |

## 5. 字段与校验

| 字段 | 编码 / 类型 | 规则 |
|---|---|---|
| 素材路径 | asset_path / 只读 | 展示存储路径 |
| 引用次数 | bind_count / 只读 | 展示素材使用范围 |
| 产品文件夹 | product_id / 只读 | 当前详情页围绕一个产品文件夹展开 |
| 类型文件夹 | asset_type / 分组 | 主图、详情图、SKU 图 |

## 6. 状态与反馈

- 当前为V1.0。
- 如果某类图片为空，保留空文件夹提示，方便后续上传补齐。

## 7. 依赖对象与接口

| 类型 | 标识 | 作用 |
|---|---|---|
| 对象 | OBJ-ASSET | 素材对象 |
| 接口 | API-ASSET-DETAIL-GET | 素材详情查询 |

## 8. 范围边界
