## 1. 机制目标

统一维护渠道商品库存，支持人工调整、同步外部渠道结果和查看库存变化记录。

## 2. 触发条件

- 库存运营在库存列表发起调整
- 库存运营触发同步
- 渠道商品详情查看关联库存

## 3. 流转规则

- 1. 库存列表按产品、渠道、店铺、SKU 聚合展示总库存、可用库存、锁定库存和可售库存。
- 2. 人工调整时记录调整量、原因、操作人和时间，并更新库存快照。
- 3. 触发同步后写入同步任务状态，等待外部渠道结果回写。
- 4. V1.0开放库存详情与库存调整记录页查看完整台账。

## 6. 异常与回退

- 库存同步失败时保留失败原因，列表中突出展示。
- 调整后可售库存小于零时，不允许保存。
- 同一库存记录存在未完成同步任务时，禁止重复发起同步。
